1. `.env` 파일을 루트에 생성하고 아래와 같이 입력해 주세요.

```
SKIP_PREFLIGHT_CHECK=true
API_END_POINT=http://13.209.30.200
CHANNEL_ID=6182ac3ce1ecd063dabf10af
```

2. `yarn` 이후 `yarn start` 입력해 주세요.
